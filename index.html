🔍 纵横分析法 (Vertical-Horizontal Analysis)

核心理念：以"行业 → 纵向演变 → 横向对标 → 估值 → 结论"5段式分析架构,系统性拆解项目投资价值。

  1. 行业分析:行业全景、技术路线、政策环境、市场规模、竞争格局
  2. 纵向演变:公司发展史、关键里程碑、战略演进、护城河形成路径
  3. 横向对标:与可比上市公司/同行对标,营收规模、盈利能力、运营效率、技术指标
  4. 估值分析:基于可比公司PE/PB/EV-EBITDA/PS + DCF辅助验证,确定估值锚点
  5. 综合结论:投资亮点、主要风险、最终判断、行动建议

数据真实性承诺:所有数据点均标注信息来源(iFind F9/EDB / MX / 行业研报 / 审计报告)和置信度(⭐⭐⭐⭐⭐ 审计级 / ⭐⭐⭐⭐ 行业研究 / ⭐⭐⭐ 公开报道 / ⭐⭐ 待补充)。所有推理过程均明示分析逻辑。
